How to add the iPhone Simulator to the Launchpad in OSX

A useful frontend development tip. By Andy Shora

← back to andyshora.com | Tweet this article

1. Create a link to the iPhone Simulator

Open your Terminal, and run the following command:

sudo ln -s /Applications/Xcode.app/Contents/Applications/iPhone\ Simulator.app /Applications

2. Drag the icon onto the Launchpad

Open your Applications folder, and drag the new 'iPhone Simulator' icon down onto the Launchpad.
